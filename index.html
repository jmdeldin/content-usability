Content Usability Toolkit

This is a Google Chrome extension to evaluate content based relevance, readability, legibility, and navigability. It is a portion of my master's project and is under heavy development.

Requirements

  1. Node (brew install node)
  2. Buster (npm install -g buster)

Testing

  1. buster-server
  2. Visit http://localhost:1111, click "Capture"
  3. buster-autotest

Running

  1. Install Extensions Reloader in Chrome.
  2. On chrome://extensions, select "Load unpacked extension"
  3. Select the project's root directory
  4. Visit a web page and select the magnifying glass
